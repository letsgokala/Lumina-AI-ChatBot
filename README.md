# Lumina AI Workspace

Lumina is a polished AI chat workspace built with React and Gemini. It focuses on a premium chat experience with persistent sessions, streaming responses, markdown rendering, image attachments, and fast navigation between conversations.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- React Query
- Google GenAI SDK
- React Markdown

## Features

- Real-time streaming chat responses
- Regenerate and stop-generation controls
- Persistent chat history with rename, search, and pin actions
- Image uploads with click-to-attach and drag-and-drop support
- Markdown rendering with copyable code blocks
- Auto-resizing composer with keyboard shortcuts
- Responsive sidebar workflow for session management

## Setup

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env` and set `GEMINI_API_KEY`
3. Start the development server:
   `npm run dev`
4. Create a production build:
   `npm run build`

## Scripts

- `npm run dev` starts the local dev server
- `npm run build` creates a production bundle
- `npm run preview` previews the production build
- `npm run lint` runs the TypeScript check
